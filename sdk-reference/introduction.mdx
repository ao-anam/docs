The Anam AI JavaScript SDK provides a simple and intuitive API to interact with Anam AI's services. It's designed to help developers integrate Anam AI's digital personas into their JavaScript applications.

## Prerequisites

### Anam API key

Your first API key will be created for you when you create an account. If you don't have your API key or if you need to create a new one, see our [API key management guide](/guides/get-started/api-key-management).
