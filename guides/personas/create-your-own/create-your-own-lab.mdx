## Creating Custom Personas with the Anam Lab
From the [Anam Lab](https://lab.anam.ai) homepage, click "Create a persona from scratch" to start designing your persona.
<Frame>
  <img src="/images/personas/create-persona-dark.png" alt="Create a persona from scratch" />
</Frame>

### Name and Description
Give your persona a name and description to help you easily identify it.
<Frame>
  <img src="/images/personas/name-description-dark.png" alt="name and description" />
</Frame>

### Personality
The persona's personality is a short description of the persona's character traits. A concise description of the persona's personality will help the LLM understand how to respond quickly to user input.
<Frame>
  <img src="/images/personas/personality-dark.png" alt="personality" />
</Frame>

### System Prompt
The system prompt is the persona's brain. It is the persona's knowledge base and instructions for how to respond to user input. Try to clearly define the persona's role and how it should respond to user input.
<Frame>
  <img src="/images/personas/system-prompt-dark.png" alt="system prompt" />
</Frame>

### Filler Phrases
Filler phrases are short phrases that the persona can use to fill in gaps in the conversation. They are used to enhance the persona's response times and make the conversation feel more natural. If you don't want to use your own phrases, you can select "use default phrases" to import a set of phrases that are commonly used.
<Frame>
  <img src="/images/personas/filler-phrases-dark.png" alt="filler phrases" />
</Frame>

### Persona Preset
The persona preset is the persona's defines your persona's appearance and voice. Select the preset from the list that best suits your brand. 
<Frame>
  <img src="/images/personas/preset-dark.png" alt="persona preset" />
</Frame>

### Confirm
Once you've filled in all the fields, click "Create" to create your persona. To view your new persona click on it from the list of personas. You can chat with it right away by clicking the "Chat" button.
<Frame>
  <img src="/images/personas/persona-details-dark.png" alt="create persona" />
</Frame>

### Using your custom persona
Once you've created your persona, you can use it in the same way as default personas, by using the Persona ID with the Anam SDK in your application. The persona ID is displayed in the "Additional Info" section on the persona's details page.
<Frame>
  <img src="/images/personas/additional-info-dark.png" alt="create persona" />
</Frame>
